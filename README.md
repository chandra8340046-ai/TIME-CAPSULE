The Time Capsule project is a web-based application that allows users to store messages for their future selves and unlock them at a specified date and time. The system simulates the concept of a traditional time capsule in a digital environment, where users can write a message, set an unlock time, and securely lock the capsule until the selected time is reached. The frontend of the project is developed using HTML, CSS, and JavaScript to provide an interactive and responsive user interface with features such as a real-time clock and countdown timer. The backend is implemented using Java, which manages the capsule data, stores messages using file handling and serialization, and checks the unlock condition based on system time. Once the specified time arrives, the stored message becomes accessible to the user. This project demonstrates key concepts of web development, time-based logic, object-oriented programming, and data persistence, making it a useful academic project for understanding the integration of frontend and backend technologies.
